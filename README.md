**Docker Alpine Tester:** to perform network tests. You can use this image in **OpenShift**.

Pre-installed programs:
-   drill
-   htop
-   bind-tools
-   wget
-   curl
-   nmap
-   mariadb-client
-   vim
-   openssl
-   bash
-   jq
-   sudo
-   iputils
-   busybox-extras
